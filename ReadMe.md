## Overview

## Build and Running Instructions
```
    git clone https://github.com/tanujthakkar/PID_Controller.git
    mkdir build && cd build
    cmake .. 
    make

    <!-- Run app -->
    ./app/pid

    <!-- Run tests -->
    ./test/cpp-test
```
